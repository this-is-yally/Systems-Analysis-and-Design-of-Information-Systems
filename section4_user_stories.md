# Розділ 4. Функціональні сценарії (User Stories)

## User Stories

1. **Як бібліотекар, я хочу додати нову книгу до каталогу, щоб читачі могли знайти її.**
2. **Як читач, я хочу знайти книгу за назвою чи автором, щоб дізнатися, чи вона доступна.**
3. **Як читач, я хочу забронювати книгу, щоб отримати її пізніше.**
4. **Як бібліотекар, я хочу зареєструвати нового читача, щоб він міг користуватися сервісом.**
5. **Як бібліотекар, я хочу переглянути список прострочених книг, щоб повідомити читачів.**

## Пріоритети

- Високий: додавання/редагування книг, видача/повернення, пошук.
- Середній: бронювання книг, історія позик.
- Низький: розширена статистика та звіти.
